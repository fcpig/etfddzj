# 大道至简 · 均衡配置助手

> 指数投资均衡配置 — 普通人也能搭建自己的养老基金
>
> Builder 赛道 · AI 投资助手 Hackathon 参赛作品

一个面向普通个人投资者的 **AI 投资助手**，以全球宽基指数 ETF 均衡配置为核心，帮助用户构建跨资产、跨市场的长期投资组合，通过 AI 解读、历史回测、智能调仓提醒，陪伴用户长期持有，减少追涨杀跌的非理性行为。

## ✨ 核心功能

- 🧠 **AI 组合配置** — 内置耶鲁模式、经典 60/40、全球均衡、A 股平衡等经典模板；支持 A股/美股切换；滑块实时调节权重；AI 通俗解读组合
- 📈 **历史回测** — 基于真实 ETF 历史数据，计算年化收益、最大回撤，绘制净值曲线
- 🔔 **智能通知与陪伴** — 市场波动时"反恐慌"解读、偏离阈值自动提醒、长期价值观推送
- ⚖️ **自动调仓** — 支持阈值提醒、定期再平衡、0 损耗投入/提取配平等多种调仓方式
- 📊 **组合总览** — 收益、偏离度、风险等级、持仓明细一屏看懂
- 💬 **AI 对话** — 随时就组合、市场、配置理念提问

## 🛠️ 技术栈

- ⚛️ React
- 🌋 火山引擎 API（大语言模型）
- 📱 移动端竖屏模拟
- 🐍 Python（历史数据处理）

## 📦 数据来源

东方财富、雅虎财经公开的 **20 支 ETF 2025 年走势**数据，经 AI 辅助 Python 脚本清洗、对齐、归一化处理。

## 🚀 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

或直接用静态服务器打开演示页面：

```bash
python -m http.server 8000
# 浏览器访问 http://localhost:8000/md-demo.html
```

## 🎯 体验路径

1. 首页点击"开始配置" → 选择账户类型（A 股/美股）
2. 选择大师模板或自定义权重 → AI 自动分析组合
3. 查看历史回测曲线 → 设置调仓规则 → 保存组合
4. 在通知页接收 AI 的波动安抚、调仓提醒与观点推送

## 💡 创新点

- 🎓 **机构配置散户化** — 把经典捐赠基金/主权基金配置直接映射到可买 ETF
- 🧘 **反人性的波动安抚** — 在市场恐慌时告诉用户"为什么不需要操作"
- 🔁 **0 摩擦配平** — 用现金流（新入金/提取）自然再平衡，降低交易成本
- 🌏 **跨市场一体视图** — A 股+美股统一敞口视角
- ⚖️ **解释先于推荐** — 每个建议都讲清楚为什么
